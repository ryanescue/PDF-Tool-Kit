Which method should I use?

Pick the option that matches the type of PDF you uploaded.

  • Auto

    Best default. Tries PyPDF first and falls back to OCR if the PDF does not expose selectable text.

  • PyPDF

    Fastest option for digitally-created PDFs (exports from Google Docs, Word, etc.). Uses the embedded text layer.

  • Tesseract OCR

    Use for scanned documents. Converts each page to an image and runs Tesseract for English characters.

  • EasyOCR

    Similar to Tesseract but better for mixed or stylized fonts. Supports multiple languages if your PDFs include them.
